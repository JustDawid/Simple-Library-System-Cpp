# Simple Library Management System 📚

A mid-level C++ console application demonstrating Object-Oriented Programming (OOP) principles, STL containers, and modern C++ features.

## 🚀 Features
* **Add Books:** Easily add new books to the library system.
* **Borrow Books:** Update the status of a book to 'borrowed'. Includes validation to check if the book is already taken.
* **Return Books:** Change the status back to 'available'.
* **View Library:** Display all books with their current statuses.

## 🧠 Technical Highlights
This project was built to practice and demonstrate:
* **OOP Concepts:** Encapsulation and Abstraction (separate `Book` and `Library` classes).
* **Standard Template Library (STL):** Usage of `std::vector` for dynamic array management.
* **Modern C++ (C++11/C++17):** * Usage of `auto` and range-based for loops.
  * Lambda expressions utilized with the `std::find_if` algorithm for searching objects within containers.
  * Constant correctness (`const` methods and pass-by-const-reference).
